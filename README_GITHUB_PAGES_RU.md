# Bahandisation GitHub Pages demo

Этот архив нужен только для быстрой демонстрации интерфейса на GitHub Pages.

Он работает без backend, поэтому настоящий Roboflow здесь не запускается. Фото можно загрузить, форму можно заполнить, заявки сохраняются в localStorage браузера, но AI-анализ работает в ручном demo-режиме.

Для настоящего Roboflow используй полный архив `BAHANDISATION_FULL_NODE_GITHUB_*.zip` и деплой через Render или Railway.

## Как загрузить

1. Создай репозиторий на GitHub.
2. Загрузи файлы из архива прямо в корень репозитория.
3. Открой `Settings` -> `Pages`.
4. Выбери `Deploy from a branch`.
5. Branch: `main`.
6. Folder: `/root`.
7. Нажми `Save`.

Если видишь 404, значит `index.html` лежит не в корне.
